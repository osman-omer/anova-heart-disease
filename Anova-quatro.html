Descriptive statistics by chest pain type

Calculate summary statistics (mean, SD, median, IQR) for cholesterol by chest pain type. Both mean/SD and median/IQR are reported because normality is uncertain; median/IQR is robust to skewed data.

Normality check: QQ plots by group

Assess normality of cholesterol within each chest pain group. QQ plots help identify deviations from normality, guiding choice between ANOVA and Kruskal–Wallis.

Code 
ggqqplot_group <- function(data, grp) {
  data %>% 
    ggplot(aes(sample = chol)) +
    stat_qq() +
    stat_qq_line() +
    facet_wrap(vars({{ grp }}), nrow = 2) +
    labs(title = "QQ Plots of Cholesterol by Chest Pain Type")
}

ggqqplot_group(data, cp)

Homogeneity of variance (Levene’s test)

Levene’s test is used to assess the homogeneity of variances across chest pain groups, which is an important assumption for parametric ANOVA.

Code 
levene_res <- data %>% 
  levene_test(chol ~ cp)

levene_res %>% knitr::kable(caption = "Levene's Test Results")
Levene’s Test Results
df1 df2 statistic p
3 1021 5.06858 0.0017323

The test shows a significant result (p = 0.00173), indicating that variances are not equal across groups. This violation supports the decision to use the non-parametric Kruskal–Wallis test instead of ANOVA.

Kruskal–Wallis test (used because assumptions violated)

Since the normality and homogeneity assumptions are violated, the Kruskal–Wallis test is appropriate for comparing cholesterol levels across chest pain groups. It is a non-parametric alternative to one-way ANOVA.

Code 
kruskal_res <- data %>% 
  kruskal_test(chol ~ cp)
kruskal_res %>% knitr::kable(caption = "Kruskal–Wallis Test")
Kruskal–Wallis Test
.y. n statistic df p method
chol 1025 8.901268 3 0.0306 Kruskal-Wallis

The Kruskal–Wallis test is significant (p = 0.0306), indicating that there is a statistically significant difference in cholesterol levels between at least two chest pain groups.

Post-hoc Dunn test with Bonferroni correction

Since the Kruskal–Wallis test indicates a significant difference, a post-hoc test is required to determine which specific groups differ. Dunn’s test with Bonferroni correction controls for multiple comparisons.

Code 
posthoc <- data %>% 
  dunn_test(chol ~ cp, p.adjust.method = "bonferroni")
posthoc %>% knitr::kable(digits = 3,
                         caption = "Dunn Post-hoc Results")

Effect size (epsilon squared)

Calculating effect size quantifies the magnitude of the difference between groups, beyond just statistical significance. For Kruskal–Wallis, epsilon squared (η²[H]) is appropriate.

Code 
ef_size <- data %>% 
  kruskal_effsize(chol ~ cp)
ef_size %>% knitr::kable(caption = "Effect Size Result")
Effect Size Result
.y. n effsize method magnitude
chol 1025 0.0057799 eta2[H] small

The effect size is very small (η²[H] ≈ 0.0058), indicating that although there is a statistically significant difference between groups, the practical impact is minimal.

Final visualization with significance stars

Adding significance stars on the boxplot highlights which group comparisons are statistically significant. This makes it easier to interpret post-hoc results visually.

Code 
stat.test <- posthoc %>% add_xy_position(x = "cp")  # determines star positions

fin_vis <- ggboxplot(data, x = "cp", y = "chol", 
                     fill = "cp", palette = "jco",
                     add = "jitter", alpha = 0.6) +
  stat_pvalue_manual(stat.test, label = "p.adj.signif", hide.ns = TRUE) + # hide non-significant
  labs(
    x = "Chest pain type", 
    y = "Cholesterol (mg/dL)",
    title = "Significant Differences in Cholesterol Levels"
  ) +
  theme_minimal()+
  theme(legend.position = "none")

The plot shows that “Typical angina” vs “Non-anginal pain” has a significant difference in cholesterol levels (p.adj < 0.05), while other comparisons are not significant. This confirms the post-hoc Dunn test results visually.

Conclusion

The Kruskal–Wallis test revealed a statistically significant difference in cholesterol levels across chest pain types (p = 0.031). Post-hoc analysis with Bonferroni correction indicated that the significant difference is mainly between “Typical angina” and “Non-anginal pain” groups. Effect size is small (ε² ≈ 0.006), suggesting that the magnitude of differences is limited. Overall, cholesterol levels vary slightly among chest pain categories, but most differences are not statistically significant. This analysis demonstrates the non-parametric approach for group comparisons when ANOVA assumptions are violated
